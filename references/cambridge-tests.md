# 剑桥雅思14-20题型对照表 (Cambridge IELTS 14-20 Question Type Reference)

## 使用说明

本文件用于根据学生薄弱题型，推荐剑14-20中的同类练习篇目。推荐时优先匹配题型，再根据难度筛选。

难度标注说明：
- 基础：适合5.0-5.5水平学生
- 中等：适合5.5-6.5水平学生
- 偏难：适合6.5-7.0+水平学生

## 推荐匹配规则

当学生某题型薄弱时，按以下规则推荐：

1. **基础水平学生**：推荐难度标注为"基础"的篇目
2. **进阶/中级水平学生**：推荐难度标注为"中等"的篇目
3. **高级/冲刺水平学生**：推荐难度标注为"偏难"的篇目
4. 每个薄弱题型推荐2-3个篇目，优先推荐近期（剑18-20）的题目
5. 如果学生已经做过某个篇目，跳过该篇目推荐下一个

**注意：** 以上题号和分布为基于剑桥雅思真题常见出题规律的参考整理。实际使用时，如果学生告知了具体做题来源，以实际为准。推荐时标注"建议核对真题确认题号"。
