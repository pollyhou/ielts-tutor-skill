# 雅思阅读高分解法 (Reading Methods)

## 核心理念

雅思阅读的本质是"信息匹配游戏"，不是翻译考试。你不需要读懂全文，只需要找到答案对应的那两三个句子。90%的答案通过同义替换呈现。

**三大核心能力：定位 + 替换 + 推断**

---

## 通用解题流程

1. **先题后文** — 先读题目，划出关键词，带着问题读文章
2. **关键词定位** — 回原文找关键词或其同义替换所在位置
3. **锁定答案句** — 在定位区域精确判断哪一句是答案句
4. **比对排除** — 将答案句与选项/题干比对，排除干扰项

**关键词选择优先级：**
专有名词（人名/地名/机构名）> 数字/日期 > 特殊符号 > 核心名词 > 动词

**时间分配原则：**
- Passage 1: 15-17分钟
- Passage 2: 18-20分钟
- Passage 3: 20-23分钟
- 留2分钟涂卡检查

---

## 各题型详解

### 1. True/False/Not Given (判断题)

**题型本质：** 考查对原文信息的精确理解与逻辑判断能力。区分"与原文矛盾"和"原文未提及"。

**解题步骤：**
1. 读题干，划出关键词（主语+谓语+宾语核心信息）
2. 回原文定位包含关键词的句子
3. 逐词比对题干与原文：
   - True：信息一致（允许同义替换，但意思相同）
   - False：信息矛盾（意思相反或有明确否定）
   - Not Given：原文无此信息（无法判断对错）
4. 特别注意限定词：all, some, always, usually, never 等

**易错陷阱：**
- 把 Not Given 选成 False：原文没说不代表错
- 把 False 选成 Not Given：只要原文有矛盾信息就是 False
- 忽略程度副词：原文 "some" vs 题干 "all" → False

**高分技巧：**
- True 通常有同义替换，False 通常有反义词或否定结构
- Not Given 常出现原文完全没有的对比关系或因果关系
- 连续题目的答案在原文中通常按顺序出现

### 2. Matching Headings (段落标题匹配)

**题型本质：** 考查段落主旨概括能力，需要区分主旨与细节。

**解题步骤：**
1. 先读所有 heading 选项，划出每个选项的核心词
2. 读段落时重点看：首句、末句、转折后的句子
3. 注意段落中的逻辑信号词：however, but, moreover, in addition
4. 排除已选选项，缩小范围

**易错陷阱：**
- 把细节当主旨：某段提到某个例子，但主旨不是那个例子
- 被 heading 中的原词误导：原文出现了某词不代表该 heading 正确

**高分技巧：**
- 段落首句是主旨的概率最高（约70%）
- 如果首句是过渡句，主旨通常在第二句或末句
- 带有 however/but 的句子往往是真正的段落重点

### 3. Matching Information (信息匹配)

**题型本质：** 在不同段落中找到特定信息的出处。

**解题步骤：**
1. 读题干，划出每个题目的核心信息词
2. 带着关键词扫读各段落
3. 找到匹配信息后标记段落字母
4. 一个段落可能对应多个题目

**易错陷阱：**
- 与 Matching Headings 混淆：信息匹配找的是具体信息，不是段落主旨
- 遗漏：一个段落可以匹配多个题目

**高分技巧：**
- 题干中的专有名词是最可靠的定位词
- 注意题目中可能出现 "reason", "example", "comparison" 等抽象词，需要找对应的信息类型

### 4. Matching Features (特征匹配)

**题型本质：** 将人物/机构/地点等与其特征或观点匹配。

**解题步骤：**
1. 先读所有 features（选项），了解要匹配什么
2. 在原文中找到所有 features 对应的主体（人名/机构名）
3. 阅读每个主体相关的句子，与选项匹配
4. 注意选项可以重复使用或不用

**高分技巧：**
- 先在原文中标出所有主体出现的位置
- 主体说的话（引号内容）通常是匹配关键
- 选项可能有干扰项，需要精确匹配

### 5. Multiple Choice (选择题)

**题型本质：** 考查对特定信息的理解和推断。

**解题步骤：**
1. 读题干，明确问什么
2. 回原文定位相关段落
3. 逐个排除选项：
   - 原词重现的选项往往是干扰项
   - 同义替换的选项更可能是正确答案
4. 注意题干中的 EXCEPT/NOT 等否定词

**易错陷阱：**
- 被原词重现的干扰项迷惑
- 选择"看起来合理"但原文没说的选项

**高分技巧：**
- 正确选项通常是对原文的同义替换或概括
- 如果两个选项都看起来对，选更贴近原文表述的
- 注意绝对化表述（always, never, all）通常是错的

### 6. Summary Completion (摘要填空)

**题型本质：** 考查对文章部分内容的概括理解和词汇识别。

**解题步骤：**
1. 先读 summary 全文，理解大意
2. 划出空格前后的关键词
3. 回原文定位对应段落
4. 在原文中找到同义替换的词填入
5. 注意词性变化和字数限制

**两种类型：**
- 带词库 (with box)：从给定词汇中选择，注意词性匹配
- 无词库 (without box)：从原文找词，可能需要变形

**高分技巧：**
- 空格前后的词是定位关键
- 带词库题先判断词性（名词/动词/形容词），缩小选择范围
- 注意语法：单复数、时态要与上下文一致

### 7. Sentence Completion (句子填空)

**题型本质：** 补充完整句子，考查信息定位和语法。

**解题步骤：**
1. 读句子，理解句意和缺失部分
2. 划出关键词回原文定位
3. 找到对应信息，提取答案词
4. 检查语法是否通顺

**易错陷阱：**
- 答案词数超出限制（注意 NO MORE THAN X WORDS）
- 忽略语法一致性

### 8. Note/Table/Diagram Completion (笔记/表格/图表填空)

**题型本质：** 考查对结构化信息的提取能力。

**解题步骤：**
1. 先看表格/图表结构，理解信息组织方式
2. 利用已给出的信息作为定位线索
3. 回原文找对应信息
4. 注意答案格式要与表格中其他项一致

**高分技巧：**
- 表格中的标题和已填内容是定位锚点
- 按表格顺序在原文中找信息，通常也是顺序的

### 9. Short Answer (简答题)

**题型本质：** 用原文中的词回答特殊疑问句。

**解题步骤：**
1. 读问题，明确问什么（What/When/Where/Who/How/Why）
2. 划关键词回原文定位
3. 找到答案后用原文原词回答
4. 注意字数限制

**高分技巧：**
- 疑问词决定了答案类型（When→时间，Where→地点）
- 答案通常在定位词附近1-2句内
