# IELTS Tutor Skill for TRAE

> 雅思私教学情分析系统 — 拍照即可生成专业学情报告

## 简介

这是一个 [TRAE](https://www.trae.ai/) Skill，专为雅思私教场景设计。学生完成练习后拍照发送，系统自动识别科目与题型，进行多维度薄弱分析，匹配剑桥雅思真题推荐，给出高分解法，并生成每日定制练习方案。

## 核心功能

| 功能 | 说明 |
|------|------|
| 拍照识别 | 自动识别听说读写四科及具体题型 |
| 7维度薄弱分析 | 题型正确率、词汇量、长难句、时间管理、定位能力、同义替换、逻辑推断 |
| 真题推荐 | 从剑桥雅思14-20中匹配同类练习，按难度分级 |
| 高分解法 | 每种题型的本质、步骤、易错陷阱、高分技巧 |
| 定制练习方案 | 7天每日计划，含具体篇目、用时和目标 |
| 多次跟踪 | 累积记录学习数据，自动生成趋势分析 |

## 文件结构

```
ielts-tutor-skill/
├── SKILL.md                        # 主文件（工作流程 + 报告规范 + 跟踪机制）
├── README.md                       # 本文件
└── references/
    ├── reading-methods.md           # 阅读9种题型高分解法
    ├── listening-methods.md         # 听力7种题型高分解法
    ├── writing-methods.md           # 写作Task1+Task2全解法
    ├── speaking-methods.md          # 口语Part1/2/3全攻略
    └── cambridge-tests.md           # 剑14-20题型对照表（按难度分级）
```

## 使用方法

1. 将 `.trae/skills/ielts-tutor/` 目录复制到你的 TRAE 项目中
2. 在 TRAE 中拍照发送雅思练习
3. 系统自动生成 HTML 学情分析报告

## 报告示例

每次分析生成一份包含以下板块的 HTML 报告：

1. **最薄弱的部分** — 雷达图 + 薄弱维度排行 + 错题逐题分析
2. **历年真题同类推荐** — 剑14-20匹配，按难度分级
3. **高分解法** — 题型本质 → 步骤 → 陷阱 → 技巧
4. **定制练习方案** — 7天每日计划

## 技术特点

- 四科全覆盖：听力、阅读、写作、口语
- 自动判水平：5个等级（基础/进阶/中级/高级/冲刺）
- 剑桥雅思14-20真题库（含剑20最新真题）
- HTML报告完全自包含，支持打印
- JSON数据跟踪，支持多次趋势分析
